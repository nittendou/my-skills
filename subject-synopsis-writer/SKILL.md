# Subject Synopsis Writer Skill

Rewrites subject synopses to be accurate, engaging, and appropriately levelled.

---

## How to Use This Skill

### Step 1 — Get the source material
User may provide: existing synopsis, syllabus, topic list, or just a description.
If only a title is given, ask for key topics before proceeding.

### Step 2 — Clarify three things (if not already clear)
1. **Target audience**: Freshmen / Diploma / Undergraduate / Working professionals?
2. **Tone goal**: Accessible · Rigorous · Both
3. **Output needed**: Synopsis only · Synopsis + 3 name options · Full rewrite with rationale

### Step 3 — Identify the core promise
State in one sentence: "By the end, students will be able to [do X] in [context Y]."

### Step 4 — Write the revised synopsis

**Length**: 80–120 words.

**Structure** (internal, not labelled):
1. Opening hook — what the subject is about in plain language
2. Core content — what students will explore
3. Relevance bridge — why it matters
4. Forward look — what students can do with this knowledge

**Tone rules**:
- Active voice throughout
- No "This subject provides an overview of…" openers
- Name technology specifically — don't say "relevant technologies"
- Freshmen: use "you" framing and analogies
- Professional: peer-level language

### Step 5 — Generate 3 name options
- Option 1: Clean and professional (catalogue-ready)
- Option 2: Student-facing and engaging
- Option 3: Forward-looking / signals contemporary relevance

---

## Output Format

```
REVISED SYNOPSIS
[80–120 words]

SUBJECT NAME OPTIONS
1. [Name] — [rationale]
2. [Name] — [rationale]
3. [Name] — [rationale]

WHAT CHANGED (optional)
[Brief explanation]
```
